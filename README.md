# SONDA Agentic AI Bootcamp 2025

![Background](./background.png)

## Resúmenes de los Laboratorios

### Laboratorio 1: AskSonda
El laboratorio **AskSonda** enseña a crear un agente conversacional especializado utilizando Watsonx Orchestrate para asistir al equipo comercial de SONDA. El agente se configura desde cero para responder exclusivamente en español, utilizando una base de conocimientos compuesta por documentación oficial de la empresa (informes financieros, presentaciones corporativas y reportes integrados). Los participantes aprenden a configurar el mensaje de bienvenida, cargar múltiples archivos de conocimiento con indexación vectorial en modo dinámico, seleccionar el modelo LLM Bedrock Sonnet 3.5, definir el comportamiento del agente mediante prompts de sistema estructurados, y añadir herramientas como el generador de PDFs (`generate_pdf_v2`) que permite crear documentos personalizados para presentaciones de ventas, todo mientras el agente mantiene respuestas claras, profesionales y completas basadas en información oficial de SONDA.

### Laboratorio 2: AskSalesforce
El laboratorio **AskSalesforce** se enfoca en crear un agente de gestión CRM que automatiza y simplifica procesos clave de ventas en Salesforce. A diferencia de crear un agente desde cero, los participantes utilizan una plantilla preconfigurada del catálogo llamada "CRM Management" que incluye herramientas predefinidas para listar, crear y actualizar cuentas, contactos, leads y oportunidades. El laboratorio guía en la personalización del agente para que responda en español mediante la adaptación del comportamiento, el cambio del modelo a Bedrock Sonnet 3.5, y la configuración de mensajes de bienvenida y prompts iniciales. Los participantes trabajan con un caso de uso práctico centrado en soluciones Smart City & Mobility para cuentas de Gobierno y Transporte, aprenden a crear nuevas oportunidades en Salesforce mediante conversación natural, y configuran guidelines (reglas de negocio) para recordar políticas corporativas como la prohibición de regalos a agentes gubernamentales, demostrando cómo integrar políticas de compliance directamente en el flujo conversacional del agente.
